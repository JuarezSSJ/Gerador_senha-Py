# 🔑 Gerador de Senhas em Python

## ✨ Sobre o Projeto

Este projeto tem como objetivo gerar senhas aleatórias e seguras, utilizando a linguagem Python e as bibliotecas **random** e **string**.

🔒 O gerador permite que o usuário defina o tamanho da senha desejada e cria uma combinação aleatória de **letras, números e caracteres especiais**.

---

## 🔧 Tecnologias Utilizadas

- Python 3+
- Biblioteca **random** (para geração de valores aleatórios)
- Biblioteca **string** (para acessar letras, números e símbolos)

---

## ⚖️ Como Usar

### 1. Clone o Repositório

```bash
git clone https://github.com/JuarezSSJ/Gerador_senha-Py.git
```

### 2. Acesse o Diretório

```bash
cd Gerador_senha-Py
```

### 3. Execute o Script

```bash
python gerador_senha.py
```

### 4. Insira o Tamanho da Senha

Ao executar o script, o usuário será solicitado a digitar o tamanho desejado da senha.

Exemplo de entrada:

```bash
Qual o tamanho da sua senha: 12
```

Saída gerada:

```bash
Sua senha gerada: G@8r!k9zT#fL
```
